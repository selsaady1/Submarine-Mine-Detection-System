Implementation Details

Core Algorithm Pipeline

Optimal PCA + Neural Network Implementation
# Optimal PCA + Neural Network Pipeline
pca = PCA(n_components=8)  # Optimal dimensionality identified
X_train_pca = pca.fit_transform(X_train)
X_test_pca = pca.transform(X_test)

model = MLPClassifier(
    hidden_layer_sizes=(100),
    activation='logistic',
    max_iter=2000,
    alpha=0.00001,
    solver='adam',
    random_state=42
)

model.fit(X_train_pca, y_train)
accuracy = model.score(X_test_pca, y_test)
